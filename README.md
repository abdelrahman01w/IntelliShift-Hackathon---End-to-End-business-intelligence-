# IntelliShift Hackathon – End-to-End Business Intelligence Platform

## Overview

IntelliShift is an end-to-end Business Intelligence solution developed during the IntelliShift Hackathon.

The objective of this project was to transform raw transactional data into actionable business insights by building a complete analytics pipeline covering data preparation, database design, ETL, machine learning, and interactive business dashboards.

Instead of focusing only on data visualization, this project demonstrates the complete Business Intelligence lifecycle—from raw data to strategic business recommendations.

---

## Business Problem

Organizations generate large volumes of transactional data every day, but data alone does not support decision-making.

The provided business case highlighted several challenges:

- Fragmented business data
- Limited visibility into organizational performance
- Manual reporting processes
- Difficulty identifying valuable customers
- Lack of actionable insights for decision makers

Our goal was to build a scalable Business Intelligence solution capable of answering business questions across sales, customers, operations, and geographic performance.

## Project Components

### Data Preparation

The raw dataset was cleaned and validated before loading it into the analytical database. Missing values, inconsistent formats, and duplicated records were handled to ensure data quality.

### Database Design

A relational database was designed using a Star Schema to improve analytical performance and simplify reporting.

The database includes:

- Fact table
- Dimension tables
- Relationships optimized for analytical queries

### ETL Pipeline

An ETL pipeline was developed to:

- Extract raw data
- Transform business attributes
- Load cleaned data into SQL Server

This provides a reproducible and structured data workflow.

### Machine Learning

A machine learning model was developed to support predictive analytics using the processed dataset.

The notebook is available under:

```
ai_models/
```

### Business Intelligence

Interactive Power BI dashboards were developed to support decision-making across multiple business areas.

The solution includes dashboards for:

- Executive Overview
- Sales Intelligence
- Marketing & Customer Analytics
- Customer Intelligence
- Operations Analysis
- Geographic Analysis
- Advanced Analytics

## Key Insights

The analysis produced several business insights, including:

- Technology is the highest revenue-generating product category.
- Consumer customers contribute more than half of total revenue.
- Approximately 80% of revenue is generated by four product sub-categories.
- Standard Class handles most orders but represents the largest opportunity for operational optimization.
- RFM segmentation identifies high-value customers and customers at risk of churn.
- Regional performance varies significantly, highlighting opportunities for targeted expansion.

---

## Business Recommendations

Based on the analysis, the project recommends:

- Increase Average Order Value through upselling and cross-selling.
- Prioritize inventory for high-performing product categories.
- Improve customer retention using RFM segmentation.
- Optimize Standard Class fulfillment processes.
- Invest in high-performing geographic regions.
- Use forecasting to support strategic planning.
